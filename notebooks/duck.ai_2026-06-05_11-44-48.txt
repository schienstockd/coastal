## What Global Optimization Actually Means

**Global optimization** in tracking means: **Find the single best assignment of all cells across a window of frames simultaneously.**

Classic example using Hungarian algorithm:

```
Frame t:     Frame t+1:
Cell A ----→ Centroid 1
Cell B ----→ Centroid 2
Cell C ----→ Centroid 3

Question: Which assignment minimizes total cost?
Option 1: A→1, B→2, C→3 (cost = 5)
Option 2: A→2, B→1, C→3 (cost = 3)  ← Global optimum
Option 3: A→3, B→2, C→1 (cost = 7)

Global optimization picks Option 2.
```

The key: **You consider all possible assignments and pick the best one globally, not greedily.**
